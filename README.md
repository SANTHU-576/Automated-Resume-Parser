# 📄 Automated Resume Parser

## Overview

An AI-powered Resume Parser built using Python, Flask, PDFPlumber, SQLite, and Regular Expressions. The application extracts candidate information from PDF resumes and stores it in a database.

---

## Features

- 📄 Upload PDF Resume
- 👤 Extract Name
- 📧 Extract Email
- 📱 Extract Phone Number
- 💻 Extract Skills
- 🎓 Extract Education
- 🗄️ Store details in SQLite
- 🌐 Flask Web Interface

---

## Technologies Used

- Python
- Flask
- PDFPlumber
- SQLite
- HTML
- CSS
- JavaScript
- Regular Expressions (Regex)
